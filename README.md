## Employee Management System
A GraphQL API built with Node.js, Apollo Server, and MongoDB.

## Tech Stack
- Node.js
- Apollo Server
- GraphQL
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Bcrypt
- Cloudinary

## Setup
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your MongoDB URI and JWT secret
4. Run `npm run dev`
5. Server runs at `http://localhost:4000/`
